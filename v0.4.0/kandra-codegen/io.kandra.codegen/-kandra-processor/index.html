class KandraProcessor(codeGenerator: CodeGenerator, logger: KSPLogger) : SymbolProcessor

KSP processor that generates a type-safe *Table object for every @ScyllaTable class.

For an entity:

@ScyllaTable("users")
data class User(@PartitionKey val userId: UUID, ...)
